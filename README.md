# Bilibili Music Player 🎵

一个基于 **C++ (Qt)** 和 **Python** 混合开发的轻量级桌面音乐播放器。支持搜索、在线解析及下载 Bilibili 音频资源。

---

## ✨ 项目特性

*   **跨平台界面**：采用 Qt 框架构建，界面简洁直观，支持动态主题切换（可选）。
*   **混合编程**：核心逻辑由 C++ 编写，保证运行效率；解析模块利用 Python 强大的库实现。
*   **在线解析**：通过集成的 Python 脚本实时解析 Bilibili 音频链接。
*   **多功能播放**：支持播放列表管理、随机播放、进度调节及音量控制。
*   **本地管理**：下载后的音频自动归档，支持本地扫描与播放。

---

## 🛠️ 技术栈

*   **GUI 框架**: Qt 6.x (C++ / Widgets)
*   **核心语言**: C++17, Python 3.x
*   **通信接口**: `Python C API`
*   **网络处理**:  Python `requests`
*   **解析引擎**: Bilibili API Wrapper

---

## 📦 打包与发布说明

本项目使用 `windeployqt` 进行依赖提取。如果你在其他机器上运行报错，请确保 `vc_redist.x64` 环境已安装，并检查 Python 动态库是否在同一路径下。
